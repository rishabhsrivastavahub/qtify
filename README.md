# QTify – Music Streaming App 🎵

## 🚀 Live Demo
https://qtify-mu-orpin.vercel.app/

## 📌 Overview
QTify is a responsive music streaming app built with React.js. It allows users to browse, search, and explore music content with smooth UI interactions.

## 🛠️ Tech Stack
- React.js
- CSS Modules
- JavaScript (ES6+)

## ✨ Features
- Dynamic carousel UI
- Search & filter functionality
- Responsive design (mobile-first)
- Optimized performance

## ⚡ Performance
- Improved load time by ~30%
- Optimized rendering using efficient component structure

## 🧠 Learnings
- Component-based architecture
- API handling & UI rendering
- Performance optimization
